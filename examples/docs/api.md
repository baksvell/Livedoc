## Функция add

Складывает два целых числа и возвращает результат.

- **Параметры:** `a`, `b` — целые числа.
- **Возвращает:** сумму.

## Функция subtract

Вычитает второе число из первого.

- **Параметры:** `a`, `b` — целые.
- **Возвращает:** разность.
